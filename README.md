# Phonebook
Phone book application

Build
-------------

#### Clone

```sh
git clone https://github.com/warapitiya/phonebook.git
```

#### Move and install
```sh
cd phonebook/
bower install & npm install
```

#### Database (maybe in a new terminal window)
```sh
mongod --dbpath $YOUR_PATH_TO_DB
```

#### Start
```sh
npm start
```
